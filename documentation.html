Setup Guide

Welcome to AI-SaaS Elite! This template is designed to be lightweight and fast. To get started, follow these simple steps:

  1. Extract the package contents to your server or local directory.
  2. The main landing page is index.html.
  3. The application dashboard is dashboard.html.
  4. All styling is handled via assets/css/style.css.
  5. All interactivity is handled via assets/js/main.js.

Hosting: Simply upload the entire folder to any static hosting provider (GitHub Pages, Netlify, Vercel) or your own FTP server.

Theme Switcher

AI-SaaS Elite comes with a built-in Dark/Light mode toggle. Here's how it works:

  • Persistence: The user's choice is saved in localStorage, so the theme persists across page reloads.
  • Implementation: The theme is controlled by a data-theme attribute on the <body> tag.
  • CSS Variables: All colors are defined as CSS variables at the root level, making customization extremely easy.
/* Example of theme customization in style.css */
:root[data-theme="dark"] {
    --bg-color: #0f172a;
    --text-color: #ffffff;
}

:root[data-theme="light"] {
    --bg-color: #f8fafc;
    --text-color: #0f172a;
}
